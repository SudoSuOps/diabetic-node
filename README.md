# DiabeticNode

**The crystal-clear breakdown of diabetic compute.** → [diabeticnode.com](https://diabeticnode.com)

Privacy-first, hardware by hardware: why the cook fleet runs on **RTX PRO 6000 Blackwell** GPUs, why
records live on **Synology** vaults, why the edge brain is a **Jetson Orin**, and how the consumer device
swarm — Apple Watch, iPhone, CGM/IoT sensors, home voice, Whisper — reaches the diabetic. What runs where,
and why. No black box.

> The one law: private health data never leaves the box. Models flow down, receipts flow up, PHI crosses never.
>
> **It takes a swarm to run the node.**
